# AideBanque 🏦✨

**AideBanque** est une plateforme Fintech premium d'aide à la décision pour le scoring de crédit bancaire. Elle combine la puissance du Machine Learning (Régression Logistique) et de l'IA Générative (**Gemini 2.5 Flash**) pour offrir des analyses rapides, précises et expertes.

## 🚀 Fonctionnalités
- **Scoring prédictif** : Algorithme ML entraîné pour évaluer l'éligibilité au crédit.
- **Expertise IA** : Rapports détaillés générés par Gemini pour chaque dossier.
- **Dashboard Analytique** : Visualisation des données avec Chart.js (Donut & Radar charts).
- **Assistant Chatbot** : IA conversationnelle pour aider les conseillers et clients.
- **Design Premium** : Interface sombre, moderne et responsive (Glassmorphism).
- **Prêt pour le déploiement** : Configuration optimisée pour Render/Heroku avec Whitenoise.

## 🛠️ Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/SenyDelamou/AideBanque.git
   ```
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```
3. Configurez votre fichier `.env` avec votre `GEMINI_API_KEY`.
4. Lancez le serveur :
   ```bash
   python manage.py runserver
   ```
